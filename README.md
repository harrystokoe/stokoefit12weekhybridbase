# Stokoe Fit · 12-Week Hybrid Base Plan

7 days a week. Strength meets endurance. 4 progressive phases building from foundational lifts and easy runs to near-max efforts and a 24km long run. Deloads built in. Results guaranteed.

**Train everything. Sacrifice nothing.**

---

## What's Inside

A fully interactive HTML training plan — no app, no subscription, no login. Open it in any browser and go.

- **12 weeks** across 4 structured phases
- **7 sessions per week** combining strength and cardio
- **RPE + Heart Rate Zone** guidance on every session
- **Built-in deload weeks** every third week
- **Progress assessments** at Week 12 including 5km time trial and 1RM tests

## The Phases

| Phase | Weeks | Focus |
|-------|-------|-------|
| 1 — Foundation | 1–3 | Movement patterns, aerobic base, RPE 6–7 |
| 2 — Build | 4–6 | Progressive overload, Z3–Z4 running, RPE 8 |
| 3 — Accumulation | 7–9 | Highest load, threshold work, 20km long run |
| 4 — Peak Base | 10–12 | Near-max strength, Z5 speed, 24km long run |

## How to Use

1. Download `12_week_hybrid_base_plan_stokoefit.html`
2. Open in any browser
3. Navigate phases and weeks using the top nav
4. Follow the daily sessions in order

No installation required.
